
## End to End Machine Learning Project

## Student Performance Prediction

This project is an end-to-end Machine Learning application that predicts a student's Maths score based on various factors such as gender, ethnicity, parental level of education, lunch type, test preparation course, reading score, and writing score.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- CatBoost
- XGBoost
- Flask
- HTML/CSS
- Docker
- AWS

## Project Workflow

1. Data Collection
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Model Training and Evaluation
5. Prediction Pipeline
6. Flask Web Application
7. Docker Containerization
8. AWS Deployment

## Run the Project Locally

Install the required dependencies:

pip install -r requirements.txt

Run the application:

python application.py

## Deployment

The application is configured for deployment on AWS using Docker.
